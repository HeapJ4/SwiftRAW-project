# Changelog

All notable changes to SwiftRAW will be documented here. The project follows [Semantic Versioning](https://semver.org/).

## [Unreleased]

## [2.0.3] - 2026-07-21

- Restore the advanced SwiftRAW implementation reconstructed from the original `app.swiftnef` APK.
- Restore onboarding, camera-brand presets, cache progress, metadata details and the zoom navigator.
- Keep the original `app.swiftnef` application ID and advance the version code to 24.
- Repair decompiler artifacts and pass the Android lint and release-build checks.

## [2.0.2] - 2026-07-21

- Publish the clean Android source tree.
- Support embedded previews from common RAW camera formats.
- Add recursive folder scanning, thumbnail caching, zoomable viewing and JPEG export.
- Keep all processing offline with no network permission.
